成为科技DMC音乐平台服务说明与协议

您好，欢迎加入成为科技DMC音乐平台（以下简称“DMC平台”）
以下6条，有利于您更好地理解这款产品：

  1. 1、DMC平台致力于协助您将原创音乐更容易地传播并获得收益。
  2. 2、申请认证成功之后，通过成为科技及其关联公司提供的DMC平台服务和其它服务，用户可在DMC平台上创建自己的作品管理账号及作品数据库、发布交易信息。
  3. 3、DMC统计分析服务通过及时的数据帮助您了解您的作品在各平台被使用和销售的情况，并且进一步帮助您分析您的作品在市场中得表现。
  4. 4、以上DMC向您提供的服务目前是免费的。如未来DMC平台向您收取合理费用，DMC平台会采取合理途径并以足够合理的期限提前通过法定程序通知您，确保您有充分选择的权利。
  5. 5、您将获得DMC平台为您开发的更多音乐推广与统计工具。
  6. 6、通过DMC平台实现的在线交易，DMC将收取最终可分配收入15%作为服务费。

基于上文的情形描述，也基于未来的共赢计划，希望您能仔细阅读下文的协议条款，以确保您能清晰地了解，未来您在DMC平台上的权益与责任，并许下属于您的承诺。如您对协议条款有任何不认可或不理解，请不要进行后续操作。

    1. 1.1.当您同意本协议的条款并按照页面提示，完成全部的帐号申请程序并通过DMC平台审核后，您才能真正入驻DMC音乐平台。

    3. 1.3.DMC平台或许会拒绝您的帐号申请。
    4. 1.4.您通过DMC平台上传您的音乐，即表示您有权且已经将音乐的信息网络传播权（包含但不限于表演者权、录音制作者权、词曲著作权）授权给DMC平台使用，经过审核后，DMC平台将在全系列产品中推广并使用您的音乐。这里的授权为非独家授权，音乐仍然归您所有，您保留授权给任何别家音乐服务商的权利，但不得影响DMC平台按本协议约定使用您作品的权利。

    6. 1.6.DMC平台并不保证您的音乐在DMC平台上的试听量、下载量与收入，而由市场情况与音乐本身受欢迎程度而定。

      2. 2.1.2.发布您的原创作品，进入DMC的正版曲库；
      3. 2.1.3.自由选择是否接受版权购买者的报价；
      4. 2.1.4.各渠道试听下载真实数据报表；
      5. 2.1.5.推广资源申请与管理工具；
      6. 2.1.6.唱片、周边、门票展示广告位。

      3. 2.3.3.优先获得第三方音乐产品中的音乐推广位；

      5. 2.3.5.向唱片公司推荐您与您的音乐作品；

      4. 2.4.4.DMC平台自主品牌的产品在其他合作方平台上的各类互联网与移动互联网应用（包括但不限于网页连接、网页播放器、浏览器插件、电脑软件、手机应用、车载设备、家庭音响终端等介质）。
    5. 2.5.您授权DMC平台在提供音乐服务的同时，使用您音乐相关的图文信息，包括但不限于照片、个人简介、专辑封面、歌词信息等。
    6. 2.6.您授权DMC平台在全球范围内使用您的音乐与相关图文信息。
    7. 2.7.您的音乐对DMC平台的授权期限为长期，在收到您的有效书面通知之前，您的授权始终有效。

  3. 第三条 收益结算和技术服务费用

    1. 3.1.您通过DMC平台实现在线交易之净收益的85%的属于您所有。就您达成的音乐交易，您需负责开具交易金额全额的发票，并且您应自行依照法律规定缴纳相关税款。
    2. 3.2.您通过DMC平台获得的收益，您授权成为科技代为收取并依照如下规则向您结算支付：
      1. 3.2.1.您最多能获得最终可分配收入总额的85%，实际按照您对歌曲的自主版权比例进行结算支付，即词著作权比例+曲著作权比例+录音制作者权比例计算（表演者权含在录音制作者权里面，因此不另外核算，但您仍应保证表演者权授权的有效性）。若歌曲的词曲版权不归您所有，或录音制作者权已经交给其他版权代理公司，DMC平台将代表您向该等权利人支付结算音乐所得的收益，而不是向您结算支付。
      2. 3.2.2.单次作品在线收听及下载的价格由版权购买者提供报价，您可以选择是否达成交易。交易将根据DMC平台提供的数据监测报表作为结算依据。
      3. 3.2.3.对于作品的版权结算比例，DMC平台将根据权利人提供的版权比例进行结算，如有调整我们会在本页面作出修改，并以站内信方式通知到您，届时您同意DMC平台依照调整后的方案进行结算支付。
      4. 3.2.4.您参加DMC平台促销活动期间期的所有收益，将按照促销后的实际收益进行结算。
      5. 3.2.5.您若为您的作品定价或促销期价格为0，则代表您自愿放弃该作品可能获得的所有收益。
      6. 3.2.6.您了解并同意，只有在您拥有实名认证的支付宝帐号并与您申请的DMC平台账户绑定认证之后，DMC平台才会主动将您的收益直接打入您的支付宝帐号。若您并未有支付宝帐号或未与DMC平台账户绑定时，只能请您通过邮件的方式申请DMC平台工作人员为您人工结帐。
      7. 3.2.7.基于技术准备及财务准备的周期，DMC平台将于2015年10月1日起，启动对您的自动结帐工作，结算的周期自您的作品登录DMC平台开始计算。
      8. 3.2.8.有关您歌曲的下载数据和其他与结算相关的数据是以DMC平台服务器记录的数据为准。
    3. 3.3.技术服务费：您通过DMC平台使用成为科技提供的存储空间、管理工具、播放系统等技术服务应向成为科技支付技术服务费。为了扶持DMC平台发展，成为科技预计在2016年12月31日前将免收技术服务费，上述免费期到期后成为科技也可能继续公布新一轮的减免政策。如果成为科技修改减免政策或公布收费方案的，我们会在本页面作出修改，并以站内信方式通知到您，在获得您的同意之后收费政策才会生效，如果您不同意收费政策，成为科技可以选择不再继续向您提供技术服务。
    4. 3.4.您了解并同意，所有本协议下的费用支付以人民币为准。
